# Endless Number Input

A simple program that allows users to enter numbers continuously. The program keeps running until the user enters `0`.

## Features

- Accepts unlimited number inputs
- Uses a loop to keep running
- Stops when `0` is entered
- Simple beginner project

## How It Works

1. Run the program.
2. Enter any number.
3. Keep entering numbers.
4. Enter `0` to end the program.

### Example

```text
Enter a number: 7
Enter a number: 15
Enter a number: 3
Enter a number: 0

Program terminated.
```
